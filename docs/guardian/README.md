# Guardian简介

基于Vue + Three.js的可视化建模工具，在3维场景中装配实体模型，例如在驱逐舰上安装主炮、鱼雷发射器、垂发系统等，每个实体模型对应一个标准化组件算法模型（采用DEVS规范），根据装配关系，自动组合成一个完整的算法模型，可直接通过[Lionet](https://guobinnew.github.io/lionet/)引擎驱动执行。
