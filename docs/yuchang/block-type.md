## Block类型

Block具有一个Type属性，用来表示Block具备什么样的行为。目前Type 主要分为：
-  Action（动作）
-  Express（表达式）
-  Control （控制）
-  Variant （变量）
-  Event （事件）
-  Markter （标记）主要供内部使用
